## Overview
**LOOX-APTAR** is a VR game development project designed to deliver an immersive brand experience. The application integrates real-time interaction, AI-generated backgrounds, and custom 3D assets to create a rich visual narrative.

## ‚ú® Solution

### 3D Asset Creation
- Modeled complementary objects to support the core environment.
- Maintained consistent style and scale across all assets.

### AI-Powered Visual Enhancement

### Interactive Presentation
- Integrated assets and visuals in Unity for real-time interactions.
- Delivered a guided experience with intuitive navigation and narrative cues.
